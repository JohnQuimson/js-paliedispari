# Palindroma

### Chiedere all’utente di inserire una parola. Creare una funzione per capire se la parola inserita è palindroma

1. Creo una funzione e lavoro li dentro

2. con un ciclo for scorro tra le lettere della parola fino a metà
3. Confronto con un if la prima lettera con l'ultima, se sono uguali mi da ritorno 'è palindroma', altrimenti 'non è palindroma'
4. Inizializzo un promp in cui chiedo all'utente la parola
5. creo una variabile in cui inserisco l'esito del return
6. stampo la variabile
